# Wonderz Play School Website

A professional, responsive website for Wonderz Play School - offering quality education from Nursery to Class V in Faridabad.

## ✨ Features

✨ **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
🎨 **Modern UI** - Clean and professional design with smooth animations
📱 **Mobile-Friendly** - Hamburger menu and optimized layout for mobile users
⚡ **Fast Performance** - Lightweight HTML, CSS, and vanilla JavaScript
🎓 **Complete Sections** - Home, About, Academics, Facilities, Testimonials, Admissions, and Contact
📧 **Contact Form** - Admission enquiry form with validation
🔗 **Social Links** - Integration with social media platforms
♿ **Accessible** - Semantic HTML for better accessibility

## ✏️ Customization

### Update Contact Information
Edit the contact section in `index.html`:
```html
<p><a href="tel:+919876543210">+91 98765 43210</a></p>
<p><a href="mailto:info@wonderzplayschool.in">info@wonderzplayschool.in</a></p>
```

### Modify Colors
Update CSS variables in `styles.css`:
```css
:root {
    --primary-color: #FF6B6B;
    --secondary-color: #4ECDC4;
    --accent-color: #FFE66D;
    /* ... other colors */
}
```
